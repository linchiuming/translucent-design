# Translucent Design Studio

A brand design studio website built with Next.js and Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

On this Codex desktop environment, you can also run:

```bat
.\dev.cmd
```

## Quality Checks

```bash
npm run lint
npm run build
```

## GitHub Pages

This project is configured for GitHub Pages at:

```text
https://linchiuming.github.io/translucent-design/
```

The workflow in `.github/workflows/deploy.yml` builds the site as a static export and deploys the `out` directory whenever changes are pushed to `main`.
